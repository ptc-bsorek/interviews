Instructions for running the project locally:
Required installations:
NPM
MongoDB (https://www.mongodb.com/try/download/community?tck=docs_server)  
 
This project contain 2 usb folders (one for the front code and another for the backend code) 
For the Angular (will be at http://localhost:4200/)
At /frontend folder ->
npm install
ng serve
----------------------------------------------------------------------
For the server (will be at http://localhost:3000/)
At /api folder ->
npm install
node app.js

When both server and an client up & running you will be able to visit the app at : http://localhost:4200/
Then you need to login/signup, and after logged in you will redirect to the page with the animation job creation process.
